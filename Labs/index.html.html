PREGUNTAS DAW

¿Cuál es la diferencia entre Internet y la World Wide Web?
El internet es una red massiva, es una infraestructura que conecta millones de ordenadores unos a otros. Mientras que el internet simplemente es la forma de accesar a dicha información.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
Get: Obtiene informacion sobre algun recurso
Head: Obtiene informacion sobre los headers si se fuera a usar un get
Post: Manda información al servidor
Put: Crea un nuevo recurso o reemplaza uno existente
Patch: Aplica alguna modificación a algun recurso ya existente
Delete: Borra el recurso especificado.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?
Se puede usar tanto el método post como el método put pero es mas seguro utilizar el método post porque este manda la información a través del cuerpo del mensaje en vez de a través del URL.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?
El método connect

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?
Es un Ok, significa que el request ha sido exitoso.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?
No, los errores 400's son por parte del usuario. El error 404 indica que un recurso no existe. Incluso, a veces un programador puede mandar este error a un usuario para que no indague donde no debe.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?
Si, los errores 500's indican que ha habido un error por lado del servidor.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.
Significa que el tag o atributo ya no es soportado por HTML 5. Existen varios tags de HTML 4 como applet, big, isindex, etc

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
HTML 5 introdujo CSS o cascading style sheets que permiten tener un documento separado tipo .css que contiene todos los elementos pertenentes al formato. Esto hace que el código sea mucho mas sencillo y ordenado.

¿Qué componentes de estructura y estilo tiene una tabla?
Estructura:
tr: table row
th: table header
td: table cell
tbody: cuerpo de la tabla
tfoot: footer de la tabla
col: columna

Estilo:
align: aliana al centro, izq o derecha
bgcolor: color del fondo
border: especifica si tiene borde o no la tabla
cellpadding: espciio entre cellwall y cellcontent
Es importante notar que ninguno de estos atributos son soportados en HTML 5

¿Cuáles son los principales controles de una forma HTML5?
Number: input tipo numerica
range: slider
date: fecha, saca un calendario para escoger
color: saca un color picker
search: campo de entrada de texto para buscar
datalist: crea una lista donde los usuarios puede seleccionar una opción y será agregada al campo de input
tel: campo para ingresar telefono
email: campo para ingresar email
url: campo para ingresar URL
